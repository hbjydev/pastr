# Pastr/Paste

The pastebin component of the larger Pastr application.


## Features

- [x] Text submission
- [ ] Text _upload_ (upload a valid MIME type file and it'll get dumped)
- [ ] Self-destruct timers
- [ ] Ownership
- [ ] Submission deletion
- [x] Index all
